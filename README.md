# Magic Expert Agents - Digital Expert Agents for SME

Multi-Agent AI system hỗ trợ doanh nghiệp vừa và nhỏ Việt Nam.

## Tech Stack
- FPT AI Inference
- Streamlit
- Multi-Agent Architecture

## How to run
1. `pip install -r requirements.txt`
2. Copy `.env.example` to `.env` and fill FPT API Key
3. `streamlit run main.py`
